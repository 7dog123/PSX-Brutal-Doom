Use Doom II for this mod.

How to use with Zandronum.

Load it like normal ,Options make sure on the Set Video mode the render is on opengl, if the render isn't then turn it to opengl, then restart (Exit out of Zandronum normally the go back into Zandronum.)

To run PSX Brutal Doom Black Edition.

BDBE_v3.2c.pk3 > PSX_BDBE.pk3 > PSX_DOOM_BASE.pk3

To Run PSX Brutal Doom.

brutalv21.pk3 > PSX_BD.pk3 > PSX_DOOM_BASE.pk3
